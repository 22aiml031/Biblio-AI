# Python environment

This project uses a local virtual environment named `.venv`.

macOS (zsh) quickstart:

```bash
# create venv (already done):
python3 -m venv .venv

# activate:
source .venv/bin/activate

# install dependencies:
pip install -r requirements.txt

# deactivate when finished:
deactivate
```

If you use an editor, configure it to use `/Users/devangpatel/BiblioAI/.venv/bin/python` as the interpreter.
